# oinone-pamirs plugin

Oinone/Pamirs aPaaS 框架研发技能集合（14 项）。安装与技能清单详见 [marketplace 根目录 README](../../README.md)。

## 编辑须知

- **权威内容只在 `skills/<name>/`**：编辑 `SKILL.md` 及其同目录辅助文件（`reference.md`、`examples.md`、`references/`、`assets/`）。
- **`codex/prompts/*.md` 是派生物**，不要手改 —— 改完 `SKILL.md` 后跑一次 `../../scripts/regen-codex-prompts.sh` 即可重新生成。
